## 🚀 빠른 시작

### 새 백준 문제 폴더 생성

```bash
# npm 스크립트 사용 (권장)
npm run create:bj 10866

# 또는 직접 실행
node scripts/create-bj-problem.js 10866
```

이 명령어는 다음을 자동으로 생성합니다:
- `Baekjoon/BJ10866/` 폴더
- `BJ10866.js` 파일 (템플릿 기반)
- `input.txt` 파일
